# Démo d'exécution symbolique avec angr

Ce dépôt contient :

- un programme C "original" (`src/prog_original.c`)
- un programme C simplifié (`src/prog_simplified.c`)
- un notebook Jupyter (`notebooks/execution_symbolique.ipynb`)
- une configuration Binder (`binder/`)

Ouvrez le notebook via Binder pour lancer la démonstration.
